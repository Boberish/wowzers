## How every open question resolved

- **Do leveling + oaths + a talent tree combine?** → YES — they ARE one artifact: the **Ledger
  Tree**. Level-points open nodes (you choose the order); deep/keystone nodes also need an
  **oath kept** (leveling + oaths meet in the middle; oaths teach AND gate).
- **What unlocks boons / keystones / modules / creeds / curios?** → All are **tree nodes**.
  Bread-and-butter = levels only; deep + keystones = levels + oath; systems' branches are lit
  by campaign **crests**. Curios stay oath-gated.
- **"Everything funnels into points you spend" vs the meta-currency ban?** → **ONE XP meter**,
  fed by every source (raid/dungeon/world/M+); points buy **access/options, never power** →
  law-legal (Law #1 & #4 held).
- **Do we even need gear?** → **KEEP it**, reframed: no-choice **drops** = the one real
  loot-moment (distinct by *agency*, zero configurator); **Market** = optional node. Nothing
  folded.
- **How much do we gate / what's the pacing shape?** → The tree holds build-craft (persistent,
  slow); the **run stays play-forward**. Two surfaces: light fast **dungeon** (<1h, mobile),
  full **raid** (1.5–3h). Density diagnosis: 1 primary in-run system + rare big-swing, StS/Hades-style.
- **Hades-style slow grind?** → CUT (was "meh"; the tree's trickle covers pacing).
- **Reward-source differentiation (raid vs M+ vs world vs dungeon)?** → All mint the one meter;
  sources feel distinct by **which oaths gate which nodes**. THEN→WHEN stays raid-only.
- **Oath-log UI?** → the Ledger Tree screen doubles as it.
- **Boss drops = boons?** → boss drops = **no-choice curios** (fortune); boons come from the
  draft; both draw the unlocked pool.
