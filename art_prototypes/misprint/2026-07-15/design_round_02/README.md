# Misprint Duelist — design round 02

Date: 2026-07-15

Status: static comparison concepts only. No character, animation, or production
direction is approved by this round.

All three concepts were generated with the built-in image-generation tool from
the existing Misprint direction board, ready-pose card, and selected gameplay
screen. Shared prompt constraints were: one athletic practical woman; exactly
one slender rapier held down/back in a neutral side-facing pose; closed solid
glove/fist shapes with no finger anatomy; animation-friendly large shapes;
flatter screenprint/cel color blocking; no action, second weapon, shield, magic,
text, logo, or watermark.

## Final comparison set

- `duelist_variant_a_simple.png` — simplest silhouette: cropped teal jacket,
  minimal seams, four-to-six-color construction, coral directional panels.
- `duelist_variant_b_split_coat_v2.png` — tailored fencer: upright split coat,
  broad cream piping/coral lining, extremely flat costume fills. This is the
  revised B; the first output was rejected and discarded for visible finger
  segmentation and an overly active stance.
- `duelist_variant_c_masquerade_runner.png` — most graphic/playful: asymmetric
  cobalt shoulder mantle, ochre waist block, geometric coral panels, larger
  decorative shapes rather than micro-detail.

The final prompt set varied only the costume/style target above while repeating
the shared constraints verbatim. B's revision additionally required smooth,
featureless black mitten silhouettes, upright resting arms, solid costume fills,
and paper grain confined to the background.
