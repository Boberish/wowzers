# Compact layered screenprint HUD — round 02

This round answers Bill's feedback that the first modular sheet felt too flat
and kept the timing instrument too far from the character theater.

- `compact_layered_screenprint_hud_v2.png` is the initial tactile-material pass.
- `compact_layered_screenprint_hud_v3_raised.png` is the corrected checkpoint:
  the full timing cluster is raised, the five combo pips are smaller and consume
  almost no height, and the duplicate five-circle motif was removed from Wind.

V3 SHA-256:
`24b27847efef00b2434d9e704e15227a74f85f03c9435d0fcff7e16cdd54d746`

The material language is layered die-cut fiberboard/paper, painted edge
laminations, inset enamel-color wells, stitched tabs, imperfect screenprint ink,
and shallow contact shadows. It is intended to belong beside the selected
painted Duelist and bright biocooling data-center plate without returning to
dark-fantasy brass or a flat border around code graphics.

This remains a UI-only concept. It does not own parry work or any runtime/shared
HUD surface. The implementation seam from `ui_round_01/README.md` still applies.
